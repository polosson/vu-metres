Vu-mètres Javascript

Affichez des Vu-mètres entièrement CSS (sans image) grâce à Javascript

  • Créez des vu-mètres simples rapidement grâce à un plugin JQuery ou un module Angular.
  • Il suffit d'un élément block (div ou autre) avec un attribut ayant la valeur à afficher.
  • Vous pouvez définir votre propre design grâce à une feuille de style comprenant 3 classes CSS.
